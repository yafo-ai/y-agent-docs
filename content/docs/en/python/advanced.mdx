## Setting Up Python

### Installation

1. **Download Python**: Visit [python.org](https://www.python.org/downloads/) and download the latest version for your operating system.
2. **Install Python**: Run the installer and make sure to check "Add Python to PATH" before clicking Install.

### Choosing an Editor

You have several options:

- **IDLE**: Comes bundled with Python, good for beginners
- **VS Code**: Free, powerful editor with excellent Python support
- **PyCharm**: Full-featured Python IDE (free Community Edition available)
