> 建议使用启动器方式

### 1.使用Y-Agent-launch启动器

使用Y-Agent-launch启动器，快捷方便，无需复杂配置，同时不会影响本机Python环境。

<Cards>
  <Card   title="Gitee下载链接" >
[点击访问](https://gitee.com/yafo-ai/y-agent-launch/releases)
  </Card>
  <Card  title="Github下载链接" >
[点击访问](https://github.com/yafo-ai/y-agent-launch/releases)
  </Card>  
</Cards>

下载完成后，解压缩，然后双击 `Y-Agent-launch.exe` 启动程序。
<Wrapper>
![](/imgs/cn/y_agent_launch.png)
</Wrapper>


首次启动项目，会自动创建Python嵌入式虚拟环境，并自动下载依赖，请耐心等待。


> 如需更复杂设置，可以使用源码方式
###  2.从源码安装启动
从Git仓库下载
<Cards>
  <Card   title="Gitee下载链接" >
```
git clone https://gitee.com/yafo-ai/y-agent.git
```
  </Card>
  <Card  title="Github下载链接" >
```
git clone https://github.com/yafo-ai/y-agent.git
```
  </Card>  
</Cards>

运行以下命令
创建conda环境
```
conda create --name y-agent python=3.11.8
```

启动conda环境
```
conda activate y-agent
```

安装依赖（注意命令所在目录）
```
cd y-agent
pip install -r requirements.txt
```
启动项目
```
python main.py
```
