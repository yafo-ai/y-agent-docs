## 介绍

Excel 知识库是使用 Excel文档 作为知识库的一种方式。知识库的优点是易于管理。

由于 Excel 里面可以存放半结构化的数据，通常适用于产品参数、产品说明、产品价格表、客户档案、等固定格式文档。

## 如何使用 Excel 知识库

### 1. 创建知识库

选择新建知识库，选择 Excel 知识库。

创建方法和文档知识库类似。同样可以设置标签。

### 2. 上传 Excel 文件

创建好知识库，下载模板文件，填写数据。

下面主要介绍一下Excel 文件的格式。

首先模板文件如下：
| 列名 |  产品1 |
| --- |  --- |
| 唯一编码 | DF00001 |
| 标题  | 东风-5C液体洲际战略核导弹 |

除了 唯一编码 和 标题 两个必填项，其他字段根据实际情况填写。
例如：
| 列名 |  产品1 |
| --- |  --- |
| 唯一编码 | DF00001 |
| 标题  | 东风-5C液体洲际战略核导弹 |
| 卖点  | 覆盖全球 |
| 速度  | 未知 |

## 向量库操作

同样 Excel文档 也可以添加到向量库中。此操作需要手工添加向量库。

从向量库删除功能目前不支持，需要删除整个文档。删除之前请注意下载备份。

这里需要说明的是，删除后再重新上传向量化，和点击修改 重新上传向量化，是有区别的。

区别是，删除后上传，文档的内部文档ID会发生变化，修改重新上传则不会变化。

这点不是很重要，但是会导致以前流程中引用过的文档的链接失效。
