---
title: 工作空间变量格式
---

角色可以输出变量到 `工作空间`，方便多个和角色同时使用一个变量。

`工作空间变量` 的 **值类型**：

| 类型名称          | 代码表示          | 说明                     |
|-------------------|-------------------|--------------------------|
| 字符串            | `String`          | 表示文本类型的数据       |
| 整数              | `Integer`         | 表示整数值               |
| 数字              | `Number`          | 表示整数或浮点数         |
| 字符串数组        | `Array<String>`   | 表示由多个字符串组成的数组 |
| 整数数组          | `Array<Integer>`  | 表示由多个整数组成的数组   |
| 数字数组          | `Array<Number>`   | 表示由多个数字组成的数组   |
