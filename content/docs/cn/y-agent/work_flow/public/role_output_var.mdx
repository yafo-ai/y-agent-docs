---
title: 角色输出变量格式
---

角色可以输出变量，提供给下游节点使用。

通过 `变量存储` - `新增变量` 为模型指定输出的变量名称，类型，并添加变量的描述，帮助模型正确填写。


<Callout> 

注意：存储位置选择 `角色自身` ，变量名称不能与工作空间变量名称相同。

</Callout>


`角色输出变量` 的 值类型：

| 类型名称          | 代码表示          | 说明                     |
|-------------------|-------------------|--------------------------|
| 字符串            | `String`          | 表示文本类型的数据       |
| 整数              | `Integer`         | 表示整数值               |
| 数字              | `Number`          | 表示整数或浮点数         |
| 布尔值            | `Boolean`         | 表示真或假的值           |
| 对象              | `Object`          | 表示键值对的集合         |
| 字符串数组        | `Array<String>`   | 表示由多个字符串组成的数组 |
| 整数数组          | `Array<Integer>`  | 表示由多个整数组成的数组   |
| 数字数组          | `Array<Number>`   | 表示由多个数字组成的数组   |
